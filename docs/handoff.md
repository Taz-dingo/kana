## 这个项目现在是什么

`kana` 是一个面向日语初学者的五十音学习项目。

它当前不是完整课程平台，而是一个聚焦在“五十音学习体验”的小型产品。方向已经明确：

- 不是单纯展示字表
- 而是做成“总览 → 详情 → 发音 → 练习”的学习闭环

## 当前代码状态

`kana` 目前是 Astro + React + Tailwind 的轻量应用，已经可以构建。

已有能力：

- 平假名 / 片假名切换
- 五十音网格展示
- 点击字符打开详情弹窗

当前主要问题：
