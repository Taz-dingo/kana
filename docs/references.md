# Kana References

## 参考页

- [五十音在线学习](https://nya.ink/50yin/index.html)

## 这页为什么值得参考

这不是因为它的视觉设计多先进，而是因为它已经把“五十音学习”里几个关键动作连起来了：

- 总览浏览
- 点击单字进入详情
- 查看字源
- 查看笔顺
- 播放发音
- 后续还衔接测验

也就是说，它展示的是一条完整的学习路径，而不是孤立页面。

## 从参考页提炼出的有效点

### 1. 总览和单字详情是天然的两层结构

用户先看全表，再点进单字深读，这是很自然的学习路径。

### 2. 字源信息有记忆价值

“字源”不是附属冷知识，而是帮助初学者把抽象假名和具体汉字形态联系起来的重要助记点。

### 3. 发音按钮应直接可用

发音不是隐藏功能。用户在浏览和详情中都可能随时想听。

### 4. 学习产品需要闭环

如果只有展示，没有测验和反馈，记忆很难巩固。

## 不建议直接照搬的点

- 旧式 Material 卡片堆叠感较重
- 视觉层级偏碎，信息密度高但不够克制
- 组件和按钮风格年代感明显
- 首屏气质更像工具页，而不是精心打磨的学习产品

## 对 `kana` 的实际启发

`kana` 应该学习这类参考页的“学习结构”，但不要复制其“旧 UI 表现”。

更具体地说：

- 学它的信息组织
- 学它的功能闭环
- 不学它的视觉语言

`kana` 应该做成更清爽、更现代、更专注字符本身的学习界面。
