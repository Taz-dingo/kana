# Kana App

`@dingo-den/kana` 是一个面向日语初学者的五十音学习项目，目标不是只做“字符表展示”，而是做成一个可浏览、可理解、可发音、可练习的学习入口。

当前代码还处在早期原型阶段，但已经有了基础的五十音切换与详情弹窗；接下来会围绕数据准确性、学习体验、发音支持和视觉重构持续推进。

## 开发

在仓库根目录运行：

```bash
pnpm dev
```

或在仓库根目录运行：

```bash
pnpm dev
```
